# コンテンツ制作ガイドライン

## 基本方針

記事、SNS、動画では、AI生成感を減らすために、一般論だけで終わらせないことを重視します。

## 入れるべき要素

- 読者の悩み
- 最初に伝える結論
- 具体的な手順
- 比較表や判断基準
- 一次情報や公式情報の確認先
- 自分の経験、判断、迷った点
- 向いている人、向いていない人
- 注意点とリスク
- 次に読む記事や無料テンプレへの導線

## 避ける表現

- 必ず稼げる
- 誰でも稼げる
- 放置で収益化
- ノーリスク
- 使っていない商品を使ったように見せる表現
- 公式情報を確認せずに価格や条件を断定する表現

## SNS・動画への展開

1本の記事から、X投稿5本、Shorts/TikTok台本1本、LINE配信1通を作ることを基本単位にします。投稿だけを増やさず、必ず記事やテンプレ導線に戻します。

## ページ更新時の確認

- titleとdescriptionがページ内容と一致しているか
- 1ページ1目的になっているか
- CTAが多すぎて読者が迷わないか
- 商品紹介では「対象」「注意点」「公式確認」が入っているか
- 記事カードと商品カードの文言が煽りすぎていないか

# ペルソナ

## 1. 副収入を作りたい会社員

- 年齢: 25〜40代前半
- 悩み: 給料だけでは不安、でも副業に使える時間が少ない
- 求めていること: 怪しくない始め方、少額から試せる方法、具体的な作業手順

## 2. ブログ・アフィリエイト挫折者

- 悩み: 記事が続かない、何を紹介すればよいか分からない
- 求めていること: AIを使った効率化、記事とSNSの連動、案件管理の仕組み

## 3. 若手エンジニア

- 悩み: 技術スキルを収入につなげたいが、発信や案件獲得が苦手
- 求めていること: GitHub Pages、GAS、Codexを使った見せ方や小さな受注導線

## 4. 小規模事業者・個人事業主

- 悩み: AIやLINE、GASに興味はあるが、何を外注すればよいか分からない
- 求めていること: 現状整理、導線設計、低コストで試せる仕組み
